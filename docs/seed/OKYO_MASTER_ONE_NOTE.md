# Okyo Master One-Note

## One sentence

Okyo turns any restaurant meal photo into a homemade copycat recipe with estimated cost savings and a shareable dupe card.

## Main app loop

```text
Scan restaurant food
→ homemade dupe
→ estimated savings
→ grocery list
→ Dupe Challenge
→ XP/badge/ranking
→ share card
```

## First MVP

- scan/upload food photo
- AI dish recognition mock
- recipe generation mock
