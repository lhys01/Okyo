# User Flows

## First scan

```text
Open app
→ Welcome
→ Goal selection
→ Scan/upload food photo
→ Loading/analyzing
→ Result summary
→ Recipe detail
→ Save/share/export grocery list
```

## Share card

```text
Result summary
→ Tap Share Dupe
→ Preview vertical card
→ Choose platform
→ Share completed
```

## Grocery list

```text
Recipe detail
→ Tap Grocery List
→ View grouped ingredients
→ Check pantry items
→ Export/copy/share list
```

## Dupe Challenge

```text
Recipe detail
→ Start Dupe Challenge
→ Mark as cooked or upload homemade photo
→ Rate match
→ Receive match score + XP + savings
→ Share before/after card
```

## Restaurant Pack

```text
Open Packs
→ Select restaurant/chain pack
→ Browse popular dupes
→ Save recipe or Start Challenge
→ Share Restaurant Pack card
```
