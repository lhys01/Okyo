# Okyo PRD Summary

## Product

Okyo is a mobile-first AI food app that lets users scan a restaurant meal and generate a homemade copycat-style recipe with estimated cost savings, grocery list, Dupe Challenge, rankings, and shareable result cards.

## One-liner

Okyo turns restaurant meal photos into homemade dupes that save money.

## Core loop

```text
Scan restaurant food
→ generate homemade dupe
→ show estimated savings
→ export grocery list
→ cook/rate the match
→ earn XP/badge/rank
→ share the result
```

## V1 must-have features

- Camera/gallery upload
- AI dish recognition
- Recipe generation
- Restaurant Copy, Budget, and Healthy modes
- Cost comparison
- Confidence score
- Grocery list export
- Saved recipe library
- Share card generation
- Money Saved Tracker
- Dupe Challenge feedback loop
- XP, badges, weekly rankings
- Static Restaurant Packs
- Freemium scan limits
- Basic premium paywall
- Analytics event tracking
- Onboarding, settings, privacy/support links

## V1 hard exclusions

Do not build in V1:

- Full social feed
- Comments
- DMs
- Full map / DupeMap
- Restaurant reviews like Yelp
- Creator marketplace
- Advanced grocery delivery partnerships
- Full macro/nutrition tracker
- AI video cooking coach
