# Curated food artwork source pool

These food images are retained source material for a possible curated discovery or restaurant-pack experience. They are intentionally not exported from `assets/food/index.ts`, so the current mobile bundle does not ship them.

Do not reconnect this directory to production navigation until the product decision includes a real content owner, honest source labels, loading/error states, and a bundle-size review. Remove individual images only after checking both runtime references and current design/product work.
