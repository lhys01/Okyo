# Kiko Mascot Assets

Real Kiko mascot PNG assets for the Okyo mobile app. These files should be used as image assets in UI components; do not replace them with emoji, SVG, or shape-based placeholders.

- `kiko-default.png` - Kiko seated in a neutral friendly pose. Use as the default mascot or empty-state fallback.
- `kiko-wave.png` - Kiko smiling and waving. Use for onboarding, welcome screens, and friendly prompts.
- `kiko-happy.png` - Kiko seated, happy, and looking upward. Use for cheerful confirmations or light positive moments.
- `kiko-thinking.png` - Kiko writing on a clipboard. Use while generating recipes, reviewing scan results, or waiting.
- `kiko-cooking.png` - Kiko in a chef hat cooking at a pan. Use for recipe creation, cooking steps, or kitchen-focused moments.
- `kiko-celebrating.png` - Kiko with star eyes holding a pencil. Use for exciting discoveries, idea moments, or playful progress.
- `kiko-success.png` - Kiko with star eyes and a pencil in a success-style pose. Use on success states or completed actions.
- `kiko-pointing.png` - Kiko pointing at a recipe-style clipboard. Use when calling attention to recipe details or next steps.
- `kiko-scanning.png` - Kiko pointing at a checklist clipboard. Use for scan progress, review states, or checklist-style flows.
- `kiko-grocery-list.png` - Kiko proudly holding a checked clipboard. Use for grocery list screens or completed grocery actions.
- `kiko-recipe.png` - Kiko holding a recipe card with food. Use for recipe results, recipe cards, or saved recipe states.
- `kiko-recipe-card.png` - Kiko holding an alternate recipe card. Use as a secondary recipe/result illustration.
- `kiko-wave-alt.png` - Alternate Kiko waving pose with a ground shadow. Use when a slightly fuller waving pose fits better.
- `kiko-side-profile.png` - Kiko seated in side profile. Use for decorative profile moments, transitions, or calm empty states.
