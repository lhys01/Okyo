# Kiko Mascot Assets

Real Kiko mascot PNG assets for the Okyo mobile app. These files should be used as image assets in UI components; do not replace them with emoji, SVG, or shape-based placeholders.

- `kiko-default.png` - Kiko seated in a neutral friendly pose. Use as the default mascot or empty-state fallback.
- `kiko-wave.png` - Kiko smiling and waving. Use for friendly Home and sharing prompts.
- `kiko-cooking.png` - Kiko in a chef hat cooking at a pan. Use for recipe creation, cooking steps, or kitchen-focused moments.
- `kiko-celebrating.png` - Kiko with star eyes holding a pencil. Use for exciting discoveries, idea moments, or playful progress.
- `kiko-scanning.png` - Kiko pointing at a checklist clipboard. Use for scan progress, review states, or checklist-style flows.
- `kiko-grocery-list.png` - Kiko proudly holding a checked clipboard. Use for grocery list screens or completed grocery actions.
- `kiko-recipe.png` - Kiko holding a recipe card with food. Use for recipe results, recipe cards, or saved recipe states.
